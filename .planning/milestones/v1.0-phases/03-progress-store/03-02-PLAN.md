## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-06 | Tampering | router.js — lastVisited used to construct window.location.hash | mitigate | The hash is set to '#/lesson/' + moduleId + '/' + lessonId; the hash router's matchRoute() uses extractParams() which compares path segment count and pattern literals — an injected path segment that doesn't match the ':moduleId/:lessonId' pattern falls through to not-found view; no innerHTML injection occurs from this value |
| T-03-07 | Spoofing | Fake lastVisited pointing to a lesson that doesn't exist | accept | matchRoute returns the lesson view params; renderLesson calls fetchLesson which returns null for non-existent lessons; renderLessonError is shown — graceful failure, no security impact |
</threat_model>
