Return value of createTerminal(container, commandHandler):
  appendOutput(text: string, color?: string): void
    — Creates a div, sets div.textContent = text (never innerHTML), sets div.style.color = color if provided,
      appends to outputList, calls scrollToBottom()
  disable(): void
    — Sets disabled = true, sets input.setAttribute('readonly', ''), sets input.style.pointerEvents = 'none',
      sets input.style.opacity = '0.4', sets promptSpan.style.opacity = '0.4'
  setPrompt(text: string): void
    — Sets promptSpan.textContent = text
  focus(): void
    — Calls input.focus()

Parameters:
  container: HTMLElement  — the #terminal-mount div; createTerminal appends termBody into it
  commandHandler: (rawInput: string) => void  — called with trimmed input on Enter keydown

DOM structure createTerminal builds inside container:
  termBody (div)
    outputList (div)
    promptRow (div, display:flex)
      promptSpan (span, green #22c55e)
      input (type=text, transparent bg, flex:1, min-height:44px, Courier New 13px)

Key behaviors from D-01, D-02, UI-SPEC C-05, I-01:
  - On Enter keydown: trim input.value; if empty return; echo as "PS PIPELINE-DC01 > {raw}"; history.unshift(trimmed); historyIndex = -1; input.value = ''; call commandHandler(trimmed); scrollToBottom()
  - On ArrowUp: prevent default; if historyIndex < history.length-1: historyIndex++; input.value = history[historyIndex]
  - On ArrowDown: prevent default; if historyIndex > 0: historyIndex--; input.value = history[historyIndex]; else if historyIndex === 0: historyIndex = -1; input.value = ''
  - termBody click: if (!disabled) input.focus()
  - input focus: set terminal wrapper border to #22c55e (find closest [data-terminal-wrapper] or use container.parentElement)
  - input blur: reset border to var(--color-border)

Styling tokens (all inline — no CSS classes except terminal-cursor-blink keyframe added in Wave 3):
  Terminal body: background #1a1a1a, font-family 'Courier New' monospace, font-size 13px, min-height 320px, max-height 480px, overflow-y auto, padding var(--spacing-md), color var(--color-text-primary)
  Prompt span: color #22c55e, white-space nowrap, user-select none, font-family 'Courier New' monospace, font-size 13px
  Input: type text, background transparent, border none, outline none, color var(--color-text-primary), font-family 'Courier New' monospace, font-size 13px, flex 1, min-height 44px, caret-color #22c55e; autocomplete=off autocorrect=off autocapitalize=off spellcheck=false
  Output line: display block, white-space pre-wrap, word-break break-all
