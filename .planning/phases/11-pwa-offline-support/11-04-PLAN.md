<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| window beforeinstallprompt → user gesture | Install prompt is intercepted and only triggered by user click — browser enforces user-gesture requirement on deferredPrompt.prompt() |
| navigator.serviceWorker controllerchange | SW lifecycle event used as secondary signal; does not expose sensitive data; only updates UI state |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-07 | Spoofing | beforeinstallprompt prompt injection | mitigate | e.preventDefault() stores the deferred event; _deferredInstallPrompt.prompt() is only called on user click (click event handler) — browser enforces user-gesture requirement; cannot be triggered programmatically |
| T-11-08 | Information Disclosure | SW controllerchange event | accept | controllerchange event carries no payload — only signals that a new SW has taken control; no sensitive data exposed |
| T-11-09 | Denial of Service | offline-indicator.js guard clause | accept | if (!container) return — graceful no-op when #top-bar div.flex is absent (e.g., test environments without full DOM); no throw, no cascade failure |
| T-11-SC | Tampering | npm/pip/cargo installs | accept | No new installs in this plan; all packages installed and audited in Plan 11-01 |
</threat_model>
