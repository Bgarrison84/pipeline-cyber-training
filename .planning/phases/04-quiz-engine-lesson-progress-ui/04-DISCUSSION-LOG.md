## Area 1: Quiz Placement & Routing

**Q:** Where does the quiz live relative to the lesson?
**Options:** Inline at lesson bottom / Dedicated route #/quiz/moduleId/quizId
**Decision:** Inline at lesson bottom — no route change; quiz-engine injects into lesson-view DOM

**Q:** Which lessons trigger a quiz?
**Options:** Only on the last lesson of a module / On every lesson that has a matching quiz file
**Decision:** On every lesson that has a matching quiz file (flexible for future content)

**Q:** How does the quiz know which lesson to attach to?
**Options:** Lesson frontmatter declares quizId / module.json maps lessonId → quizId
**Decision:** Lesson frontmatter declares `quizId: '01'` — zero changes to module.json

## Area 2: Answer Submission UX

**Q:** How does a learner submit their quiz answer?
**Options:** Click answer card to reveal immediately / Select answer then click Submit
**Decision:** Click-to-reveal immediately — no submit button

**Q:** After answering, can the learner change their answer?
**Options:** No — locked after first click / Yes — allow retry until correct
**Decision:** Locked after first click — matches last-attempt-only storage

**Q:** For multi-question quizzes, how does the learner move through questions?
**Options:** All questions visible at once / One question at a time with Next button
**Decision:** All questions visible at once — score tallied when last question answered

**Q:** How is correct vs. wrong visually distinguished?
**Options:** Green border + check icon for correct; red border + X icon for wrong / Background fill approach
**Decision:** Green border (#22c55e) + check icon; red border (--color-destructive) + X icon

## Area 3: Progress Bar Design

**Q:** What does the sidebar progress indicator look like per module?
**Options:** Thin progress bar below module title / Fraction text alongside title / Both
**Decision:** Thin 4px horizontal progress bar below module title

**Q:** What counts as a lesson being 'complete' for progress calculation?
**Options:** Visited = progress; quiz passed = extra / Only quiz pass counts
**Decision:** Only quiz pass counts — visiting alone doesn't count toward progress

**Q:** How is a module marked 'complete' in the sidebar?
**Options:** Check icon + accent color on module name / Full bar + 'Complete' badge
**Decision:** Check-circle Lucide icon + accent color on module name

**Q:** What does the module overview page show for progress?
**Options:** Progress bar + per-lesson status list / Just a progress bar
**Decision:** Progress bar + lesson list with visited/quiz-passed status per lesson

## Area 4: Score Display on Revisit

**Q:** Where does 'You scored 3/4' appear on revisit?
**Options:** Top of quiz section / In lesson header / Only on module overview page
**Decision:** Score banner at top of quiz section — "Your last attempt: {score}/{total} correct — {date}"

**Q:** On revisit, are quiz questions shown again (locked) or hidden?
**Options:** Shown and locked with prior answers highlighted / Hidden — only score shown
**Decision:** Shown and locked — questions display in correct/wrong state; note: specific option chosen is NOT stored (D-04 Phase 3), so only the score is restored, not which answer was selected
